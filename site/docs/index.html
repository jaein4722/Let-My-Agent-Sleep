Start long jobs. Stop polling. Resume supported sessions.

Let My Agent Sleep is a small agent handoff layer for training, evaluation, preprocessing, benchmarks, migrations, and batch jobs that would otherwise make an AI agent poll logs or repeatedly continue.

npx let-my-agent-sleep install
