Claude Code

Claude Code support is experimental.

LMAS keeps long-running jobs under tmux so Claude can stop polling and continue when the work is finished.

Install

npx let-my-agent-sleep install --agent claude

Restart Claude Code after installation, then ask it to use /let-my-agent-sleep for a long-running command.

What to expect

  1. Claude starts the command through LMAS and ends the turn instead of polling.
  2. The real command continues under tmux.
  3. If Claude remains open, LMAS can wake the same session when the command finishes.
  4. If Claude closes, the command still continues and LMAS attempts a recovery resume.

Current limitation

Claude Code support remains experimental. Automatic wake-up depends on Claude still being available. When automatic delivery is not possible, LMAS keeps a recovery prompt with the run results. LMAS installs no daemon, executable wrapper, or PATH change.
